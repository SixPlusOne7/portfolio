Kinematic Analysis of Undulatory Swimmers for Bio-inspired Robotics

Python tool to quantify swimming mechanics and guide energy-efficient robot design

I wrote this code to analyze video frames of undulatory swimmers, such as fish and bio-inspired robots, to study their body shapes and movement patterns. By tracking features such as head angle and swimming speed, the code can be used to better understand underwater locomotion and support the development of robotic models that mimic efficient biological movement.

  • Body Shape Extraction: Track body midlines by marking points from head to tail.
  • Curve Generation: Convert points into smooth curves using interpolation.
  • Movement Analysis: Measure head angle and speed to study movement efficiency.
  • Data Visualization: Plot curves and data points to compare body shapes.

Tools: Python, OpenCV, SciPy, Matplotlib
